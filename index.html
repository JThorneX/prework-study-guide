Prework Study Guide

HTML

  • Foundation of a website.
  • The head element contains metadata information about the webpage.
  • The boy element represents the visible content shown to the user.
  • Start with h1 and work up numerically for headers.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Period preceding selector designates it as a class selector which can be assigned to specific elements.
  • Selecting colors with hex # is allowed.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Start with cd before navigating to where you want to go.
  • Code . to open in VS code, explorer . to open location in file explorer.
  • git add -A to stage changes, git commit -m "message" to commit, git push origin feature/branch-name to push to GitHub.
  • git pull origin main to make sure the current branch is up to date before opening a pull request.
